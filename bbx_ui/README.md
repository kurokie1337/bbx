# 🚀 BBX Workflow Studio

Desktop UI application for BBX workflows.

## Features

- 📝 Visual workflow editor with syntax highlighting
- ▶️ Run workflows with live output streaming
- ✓ Built-in validation
- 📁 File management
- 🎨 Modern dark theme UI

## Run

Windows:
```
run_bbx_studio.bat
```

Linux/Mac:
```
./run_bbx_studio.sh
```

Or directly:
```
python bbx_ui/app.py
```

## Screenshots

- Left sidebar: Quick actions
- Main editor: Edit .bbx files
- Output tab: See execution results in real-time
