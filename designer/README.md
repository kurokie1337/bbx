# BBX Visual Workflow Designer

React-based visual workflow builder.

## Features

- Drag-and-drop workflow creation
- Real-time validation
- Export to .bbx YAML
- Live workflow execution
- Template library

## Run Locally

\`\`\`bash
cd designer
npm start
\`\`\`

## Access via BBX API

\`\`\`bash
python api_server.py
# Open http://localhost:8000/designer
\`\`\`

## Docker

\`\`\`bash
docker-compose up designer
# Open http://localhost:3000
\`\`\`
